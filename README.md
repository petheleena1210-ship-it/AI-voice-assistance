AI Voice Assistant
Overview

AI Voice Assistant is a Python-based intelligent virtual assistant that interacts with users through voice commands. It uses Speech Recognition and Text-to-Speech technologies to understand spoken instructions and provide spoken responses. The assistant can perform various tasks such as searching the web, opening applications, providing date and time information, and answering user queries.

🔊 Text-to-Speech Responses
🌐 Web Search Functionality
🖥️ Open Applications and Websites

Installation
Clone the Repository
git clone https://github.com/yourusername/AI-Voice-Assistant.git
cd AI-Voice-Assistant
Install Dependencies
pip install -r requirements.txt
Required Libraries
pip install SpeechRecognition pyttsx3 pyaudio
Usage

Run the project using:

python main.py

After launching, the assistant will listen for voice commands and respond accordingly.
